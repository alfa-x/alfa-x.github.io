Cara Membuat Github Pages

Pertama-tama kita git bash di direktori yang ditentukan, kemudian kita init.

Kemudian kita gunakan git status untuk mengecek status. 

Setelah itu kita git add . yang bermaksud untuk meng add semua file di direktori tersebut

kemudian kita lakukan first commit dengan git commit -m 

kemudian kita remote repository github dengan menggunakan git remote add origin https://github.com/alfa-x/alfadani.github-io 

dan yang terakhir kita push file tadi ke repository github kita
